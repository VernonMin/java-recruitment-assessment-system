# 系统使用说明

## 1. 说明

本文档说明 MVP 阶段各角色如何使用系统。
系统归属：昱易昇（成都）科技有限公司。

支持角色：

- 候选人
- 面试官
- 招聘专员
- 管理员

前端入口说明：

- `求职者端`：只给候选人使用
- `企业端`：给面试官、招聘专员、管理员共用
- 登录页为独立页面，未登录时不显示业务导航

当前管理员默认账号：

- `admin / admin`：管理员，进入企业端

## 2. 候选人使用说明

### 2.1 登录

1. 打开系统登录页
2. 输入账号和密码
3. 点击“登录”
4. 登录成功后自动进入求职者端

说明：

- 前端默认通过 `web/config.js` 中的配置访问后端
- 正式环境下候选人不需要手工填写后端服务地址

### 2.2 退出登录

1. 点击页面右上角“退出登录”
2. 系统清理当前会话
3. 页面自动回到登录页

### 2.3 开始测评

1. 打开已分配测评列表
2. 选择一个可参加的测评
3. 阅读测评说明
4. 如果有要求，则授权摄像头
5. 如果有要求，则进入全屏
6. 点击“开始测评”

### 2.4 答题

答题行为说明：

- 选择题可直接勾选
- 填空题使用文本输入
- 简答题使用文本输入
- 场景分析题支持较长文本输入
- 系统应周期性自动保存答案

### 2.5 提交

1. 点击“提交”
2. 确认提交
3. 等待提交成功提示
4. 如果超时，系统应自动提交

### 2.6 查看结果

1. 进入已完成测评列表
2. 打开某次测评记录
3. 在招聘专员或管理员开放后查看评分与评价

## 3. 面试官使用说明

### 3.1 管理题目

1. 登录企业端
2. 进入题库管理
3. 如需快速初始化，可点击“导入示例题目”
4. 也可以手工创建题目
5. 填写题型、题干、答案、分值、解析
6. 保存为草稿或发布

当前版本说明：

- 题型支持单选、多选、判断、填空、简答、场景分析
- 已支持按关键词、题型、状态检索题目
- 已支持修改题目与删除未被引用的题目
- 选择题选项使用逐行录入，格式为 `A|选项内容`
- 已内置一组 Java 招聘示例题，覆盖 Java 基础、集合、并发、Spring 事务、后端场景题
- 示例题依据 Oracle Java 官方文档和 Spring Framework 官方文档整理，可直接用于初始演示和笔试任务配置

### 3.2 创建试卷模板

1. 进入试卷模板管理
2. 创建新模板
3. 从已发布题目中选择要纳入模板的题目
4. 调整题目分组、顺序与模板内分值
5. 保存模板
6. 如需修改，可在模板列表中打开“编辑模板”

### 3.3 评估主观题

1. 登录企业端
2. 打开某个笔试任务的提交列表
3. 筛选待评估主观题
4. 查看候选人答案
5. 如已启用 AI 建议判分，可先点击“AI 建议判分”
6. 人工复核建议分数与评语
7. 保存评估结果

### 3.4 查看结果

1. 打开试题结果页
2. 查看总分、客观题分数、主观题分数和异常事件
3. 如有需要可导出结果

## 4. 招聘专员使用说明

### 4.1 管理候选人

1. 使用管理员账号或招聘专员账号进入企业端
2. 管理员在“用户管理”中创建求职者账号
3. 管理员或招聘专员在“用户管理”中将候选人分配到笔试任务
4. 跟踪邀请状态与完成状态

### 4.2 发布笔试任务

1. 登录企业端
2. 进入笔试任务管理
3. 基于试卷模板创建笔试任务
4. 设置测评时间窗口与时长
5. 配置是否要求摄像头或全屏
6. 发布试题

### 4.3 查看推荐结论

1. 打开试题结果页
2. 查看分数、评语与异常监控事件
3. 标记推荐结论，例如“待定”或“淘汰”
4. 导出汇总结果

## 5. 管理员使用说明

### 5.1 管理用户

1. 登录企业端
2. 进入用户管理
3. 创建候选人、面试官、招聘专员、管理员账号
4. 为新账号设置初始密码
5. 查看当前账号列表和角色

当前版本说明：

- 已支持创建账号
- 已支持批量导入账号
- 已支持按关键词、角色、状态检索账号
- 已支持分页浏览用户检索结果
- 已支持编辑账号、禁用账号、重置密码
- 已支持删除未被引用的账号
- 已支持单个或批量把候选人分配到笔试任务
- 已支持在人工评估页调用 DeepSeek 兼容接口生成主观题建议分数

### 5.2 审计试题

1. 进入笔试任务管理后台
2. 先通过检索栏筛选目标任务，再用分页浏览结果
3. 使用“新增任务”按钮打开弹窗创建新任务
4. 使用任务卡片中的“编辑任务”或“修改任务”按钮打开弹窗更新已有任务
5. 查看任务状态与提交数量
6. 如有需要批量分配候选人
7. 查看异常监控事件
8. 导出报表

## 6. 监控与权限

### 6.1 摄像头

如果某场测评要求摄像头权限：

- 浏览器会向候选人发起授权请求
- 前端会周期性抓拍
- 图片会上传保存，用于事后审查

MVP 阶段这里不是实时视频监考。

### 6.2 全屏与页面行为

如果启用了相关规则，系统会记录：

- 退出全屏
- 页面失焦
- 网络断开与恢复

这些事件可能会影响本次提交的风险等级。

## 7. 常见问题

### 7.1 无法登录

请检查：

- 账号和密码是否正确
- 账号是否被禁用
- 浏览器是否允许 Cookie

### 7.2 摄像头授权失败

请检查：

- 浏览器是否允许摄像头权限
- 操作系统是否允许浏览器访问摄像头
- 是否有其他程序占用了摄像头

### 7.3 提交失败

请检查：

- 网络是否稳定
- 登录会话是否过期
- 测评时间是否已经结束

### 7.4 为什么题库里没有题

请检查：

- 是否已使用管理员或面试官账号登录

### 7.5 AI 建议判分为什么不能用

请检查：

- Worker 是否已配置 `AI_REVIEW_ENABLED=true`
- 是否已配置 `AI_REVIEW_API_KEY`
- `AI_REVIEW_BASE_URL` 与 `AI_REVIEW_MODEL` 是否正确
- 当前登录账号是否为面试官、招聘专员或管理员
- 是否已经在“题库管理”页面点击“导入示例题目”
- 是否是首次部署，还没有人工创建或导入题目

## 8. 浏览器建议

推荐使用：

- 最新版 Chrome
- 最新版 Edge

不建议在较旧浏览器中参加测评。
