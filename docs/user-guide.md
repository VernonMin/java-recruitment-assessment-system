# 系统使用说明

## 1. 说明

本文档说明 MVP 阶段各角色如何使用系统。

支持角色：

- 候选人
- 面试官
- 招聘专员
- 管理员

## 2. 候选人使用说明

### 2.1 登录

1. 打开系统登录页
2. 输入账号和密码
3. 点击“登录”
4. 登录成功后进入候选人首页

说明：

- 前端默认通过 `web/config.js` 中的配置访问后端
- 正式环境下候选人不需要手工填写后端服务地址

### 2.2 开始测评

1. 打开已分配测评列表
2. 选择一个可参加的测评
3. 阅读测评说明
4. 如果有要求，则授权摄像头
5. 如果有要求，则进入全屏
6. 点击“开始测评”

### 2.3 答题

答题行为说明：

- 选择题可直接勾选
- 填空题使用文本输入
- 简答题使用文本输入
- 场景分析题支持较长文本输入
- 系统应周期性自动保存答案

### 2.4 提交

1. 点击“提交”
2. 确认提交
3. 等待提交成功提示
4. 如果超时，系统应自动提交

### 2.5 查看结果

1. 进入已完成测评列表
2. 打开某次测评记录
3. 在招聘专员或管理员开放后查看评分与评价

## 3. 面试官使用说明

### 3.1 管理题目

1. 进入题库管理
2. 创建题目
3. 填写题型、题干、答案、分值、解析
4. 保存为草稿或发布

当前版本说明：

- 题型支持单选、多选、判断、填空、简答、场景分析
- 选择题选项使用逐行录入，格式为 `A|选项内容`

### 3.2 创建测评模板

1. 进入测评模板管理
2. 创建新模板
3. 从题库选择题目
4. 调整顺序与分值
5. 保存模板

### 3.3 评估主观题

1. 打开某个招聘场次的提交列表
2. 筛选待评估主观题
3. 查看候选人答案
4. 填写分数与评语
5. 保存评估结果

### 3.4 查看结果

1. 打开场次结果页
2. 查看总分、客观题分数、主观题分数和异常事件
3. 如有需要可导出结果

## 4. 招聘专员使用说明

### 4.1 管理候选人

1. 进入候选人管理
2. 创建或导入候选人账号
3. 将候选人分配到招聘场次
4. 跟踪邀请状态与完成状态

### 4.2 发布招聘场次

1. 进入招聘场次管理
2. 基于测评模板创建场次
3. 设置测评时间窗口与时长
4. 配置是否要求摄像头或全屏
5. 发布场次

### 4.3 查看推荐结论

1. 打开场次结果页
2. 查看分数、评语与异常监控事件
3. 标记推荐结论，例如“待定”或“淘汰”
4. 导出汇总结果

## 5. 管理员使用说明

### 5.1 管理用户

1. 进入用户管理
2. 创建或编辑账号
3. 分配候选人、面试官、招聘专员、管理员角色
4. 根据需要禁用或锁定账号

### 5.2 审计场次

1. 进入招聘场次管理后台
2. 查看场次状态与提交数量
3. 查看异常监控事件
4. 导出报表

## 6. 监控与权限

### 6.1 摄像头

如果某场测评要求摄像头权限：

- 浏览器会向候选人发起授权请求
- 前端会周期性抓拍
- 图片会上传保存，用于事后审查

MVP 阶段这里不是实时视频监考。

### 6.2 全屏与页面行为

如果启用了相关规则，系统会记录：

- 退出全屏
- 页面失焦
- 网络断开与恢复

这些事件可能会影响本次提交的风险等级。

## 7. 常见问题

### 7.1 无法登录

请检查：

- 账号和密码是否正确
- 账号是否被禁用
- 浏览器是否允许 Cookie

### 7.2 摄像头授权失败

请检查：

- 浏览器是否允许摄像头权限
- 操作系统是否允许浏览器访问摄像头
- 是否有其他程序占用了摄像头

### 7.3 提交失败

请检查：

- 网络是否稳定
- 登录会话是否过期
- 测评时间是否已经结束

## 8. 浏览器建议

推荐使用：

- 最新版 Chrome
- 最新版 Edge

不建议在较旧浏览器中参加测评。

## 9. 文档维护规则

当以下内容发生相关变更时，必须同步更新本说明文档：

- 登录流程
- 角色能力
- 测评参与步骤
- 评估流程
- 监控行为
